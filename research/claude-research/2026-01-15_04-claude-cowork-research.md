## Overview

**Product**: Claude Co-Work
**Release Date**: January 12, 2026
**Publisher**: Anthropic
**Purpose**: Claude Code for non-coders

## What It Does

Claude Co-Work brings Claude Code's powerful agent capabilities to non-technical users through:

- **Folder-based sandbox**: Work within a designated project folder
- **Progressive disclosure**: Simple interface that reveals complexity as needed
- **Natural language workflows**: Describe what you want, Claude figures out how

## Key Features

| Feature | Description |
|---------|-------------|
| **Sandbox Mode** | Safe execution environment within user's folder |
| **Progressive UI** | Starts simple, reveals advanced options as needed |
| **Non-Coder Focus** | Abstracts away technical complexity |
| **File Awareness** | Understands context from project files |

## Target Users

- Product managers who need working prototypes
- Designers who want to iterate on code
- Business analysts automating reports
- Content creators managing static sites
